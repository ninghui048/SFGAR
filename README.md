# Source-Free Graph Foundation Model Adaptation via Pseudo-Source Reconstruction

This repo contains the implementation of the model proposed in "Source-Free Graph Foundation Model Adaptation via Pseudo-Source Reconstruction".

## Environment

+ python 3.11
+ pytorch 2.0.0
+ torch_geometric 2.5.3


## Dataset
Put the datasets to the data folder.

## Run

To run the project as follows:
```
python main.py
```
